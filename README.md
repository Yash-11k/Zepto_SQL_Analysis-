# Zepto_SQL_Analysis-
MySQl
This project focuses on analyzing Zepto product dataset using MySQL.
The objective is to clean, explore and generate meaningful business insights from raw product data.


Key Analysis Performed

1)Stock availability analysis
2)Category wise product count
3)Discount analysis
4)Duplicate product detection
5)Zero MRP removal


Key Insights

1)Majority of products are in Grocery category.
2)Some products had zero MRP and were cleaned.
3)Limited products had high discount percentages.
